# Rice Plant Disease Detection Using Deep Learning

## Description
This project focuses on the automatic detection of rice plant diseases using Deep Learning techniques.
The system analyzes images of rice leaves and classifies them into different disease categories, helping farmers identify diseases early and take appropriate measures.

## Features
- Automatic disease detection
- Rice leaf image classification
- Deep learning-based prediction
- User-friendly interface

## Diseases Detected
- Bacterial Leaf Blight
- Brown Spot
- Leaf Smut
- Healthy Leaf

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib


## Objectives
- Detect rice plant diseases automatically from leaf images.
- Improve the accuracy of disease diagnosis using Convolutional Neural Networks (CNN).
- Provide a quick and efficient solution for farmers and agricultural experts.

## Features
- Upload rice leaf images.
- Automatic disease classification.
- Predicts different rice plant diseases.
- User-friendly interface.
- Fast and accurate detection.


## Dataset
The dataset contains images of healthy and diseased rice leaves collected from publicly available sources.

## Model
- Convolutional Neural Network (CNN)
- Transfer Learning (Optional: MobileNetV2, EfficientNetB0)

  ## Future Enhancements
- Disease severity prediction
- Nutrient deficiency prediction
